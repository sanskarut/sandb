# Multi Mongo Cluster

A lightweight multi-database MongoDB connection manager with load balancing, failover, and distributed query support.

---

## 📦 Installation

```bash
npm install multi-mongo-cluster
```

---

## 🤝 Contributing

```bash
git clone https://github.com/your-username/multi-mongo-cluster.git
cd multi-mongo-cluster
npm install
npm run build
```
