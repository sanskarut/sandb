# Multi Mongo Cluster

A lightweight multi-database MongoDB connection manager with load balancing, failover, and distributed query support.

---

## 📦 Installation

```bash
npm install sandb
```

---

## 🤝 Contributing

```bash
git clone https://github.com/sanskarut/sandb.git
cd sandb
npm install
npm run build
```
